## Project Overview
This is a classic Tic-Tac-Toe game for two players. Players take turns to place either "X" or "O" on a 3x3 grid. The objective is to get three of the same symbol in a row, either horizontally, vertically, or diagonally.

The game includes:
- Alternating turns between Player X and Player O.
- Detection of a win or draw.
- Reset and start new game options.

This project was created to demonstrate core web development skills on the GUVI platform.
